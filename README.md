# dbds-project

## Prerequisites

- Node version 18.x.x
- npm version 9.5.x

## Steps to start the server
- Install Nodejs and npm
- Run `cd dbds-project`
- Run `npm install --save`
- Change Database Configuration if required
  - Configurations are present in `config/db.config.js`
- Run `node index.js`
- Goto http://localhost:3000
